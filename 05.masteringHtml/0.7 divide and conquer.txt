Grouping different HTML elements can make your pages faster to load and easier to customize and maintain.

 

In this lesson, you’ll learn to group related elements.

The <div> element is a container for HTML elements that keeps your pages organized
<div>
	<!-- hello-->
	<!--hi-->
</div>

The <div> element  always takes up the full width available. This means the <div> element is block-level element

<div> is a block-level element. This means it… starts in new line

<div>
	<h2> Heading </h2>
	<p> paragraph</p>
	<img src = "url">
</div>

<div> is a container that groups related content on a web page, such as a sidebar or a navigation menu. <div> doesn’t add meaning to the content because it’s a…
non- semantic tag 


You can use <div> to apply the same style to a group of elements
